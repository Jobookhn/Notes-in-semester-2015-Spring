一、Activities

1.创建Activities

  • 创建Activity的子类
  • 实现各类回调方法
    在状态转换时调用的方法

onCreate()

  • 在创建活动时被调用
  • 做一些初始化的工作

onPause()

  • 在应用失去焦点时调用
  • 调用onPause方法并不代表活动不可见
  • 通常保存当前状态以便后续恢复
